# Tic-Tac-Toe Web Application
A Tic-Tac-Toe Web Application developed using HTML, CSS, and JavaScript. The application features a responsive and user-friendly interface with dynamic game status updates.  This project demonstrates frontend web development concepts such as DOM manipulation, event handling, game logic implementation, and interactive UI design using JavaScript.

## Features

- **Interactive Gameplay** - Play against another player in real-time
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Real-time Game Status** - Dynamic updates showing whose turn it is and game results
- **Game Reset** - Easy restart functionality to play multiple rounds
- **User-Friendly Interface** - Clean, intuitive design for an enjoyable experience

##  Technologies Used

- **HTML** - Semantic markup for game structure
- **CSS** - Styling and responsive layout
- **JavaScript** - Game logic, DOM manipulation, and event handling
